ng-set-attribute directive
=========================

How to use:
-----------

```html
<div ng-set-attribute="{'data-attr-name': ['attrvalue', true], 'data-attr-name2': ['attrvalue2', true]}">
  Text
</div>
```

```html
<div ng-set-attribute="{'data-attr-name': 'attrvalue', 'data-attr-name2': 'attrvalue2'}">
  Text
</div>
```
